# Анализ тональности русскоязычных текстов

**Датасет:** [MonoHime/ru_sentiment_dataset](https://huggingface.co/datasets/MonoHime/ru_sentiment_dataset) — русскоязычные отзывы с метками тональности.

## Структура

| Файл | Часть задания |
|---|---|
| `ru_sentiment_eda.ipynb` | **Часть 1** — EDA, тематическое моделирование (LDA), визуализация тем, варианты улучшения |
| `ru_sentiment_classifier.ipynb` | **Часть 2** — обучение классификатора (rubert-tiny2), оценка качества и скорости на CPU/GPU, варианты улучшения |
| `ru_sentiment_optimization.ipynb` | **Часть 3\*** — оптимизация модели для инференса на CPU, сравнительная таблица |
